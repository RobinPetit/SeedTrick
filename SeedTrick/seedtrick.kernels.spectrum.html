seedtrick.kernels.spectrum module¶

class seedtrick.kernels.spectrum.SpectrumKernel(k, bool normalized)¶

Bases: seedtrick.kernels.base.Kernel

Implementation of the \(k\)-Spectrum Kernel defined in [1]

Definition

Let \(\Sigma\) be an alphabet (typically the set of Amino Acids). For \(k > 0\), let \(\Phi\) be the mapping:

\[\Phi^k : \Sigma^* \to {\mathbb N}^{|\Sigma|^k} : s \mapsto (\phi_a^k(s))_{a \in \Sigma^k},\]

where for every \(a \in \Sigma^k\), \(\phi_a^k\) is the mapping:

\[\phi_a^k : \Sigma^* \to \mathbb N : s \mapsto \left|\left\{i \in \{1, \ldots, |s|-k+1\} : s_is_{i+1}\ldots s_{i+k-1} = a\right\}\right|.\]

The \(k\)-Spectrum Kernel is then defined by:

\[\kappa_k : \Sigma^* \times \Sigma^* \to \mathbb R : (s, t) \mapsto \langle \Phi^k(s), \Phi^k(t) \rangle.\]

References

[1] Leslie, C., Eskin, E., & Noble, W. S. (2001). The spectrum kernel: A string kernel for SVM protein classification. In Biocomputing 2002 (pp. 564-575).
