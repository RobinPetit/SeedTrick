class seedtrick.kernels.string_kernels.StringSubsequenceKernel(unsigned int N, float lambda_=.9, bool normalized=True)¶

Bases: object

Implementation of the String Subsequence Kernel (SSK) defined in [1].

Definition

Let \(\Sigma\) be an alphabet. For \(n \geq 0\), let \(\Sigma^n\) be the set of all finite strings of length \(n\) on \(\Sigma\).

The SSK is defined as follows:

\[\mathsf{SSK}_N : \Sigma^* \times \Sigma^* \to \mathbb R^+ : (s, t) \mapsto \sum_{u \in \Sigma^N}\sum_{\mathbf i : u = s[\mathbf i]}\sum_{\mathbf j : u = t[\mathbf j]}\lambda^{\ell(\mathbf i) + \ell(\mathbf j)}\]

References

[1] Lodhi, H., Saunders, C., Shawe-Taylor, J., Cristianini, N., & Watkins, C. (2002). Text classification using string kernels. Journal of Machine Learning Research, 2(Feb), 419-444. https://papers.nips.cc/paper/1869-text-classification-using-string-kernels.pdf
